# EcomExpert Store

![Store Screenshot](assets/store-screenshot.png)

A modern eCommerce store built on Shopify, designed to deliver a clean, fast, and enjoyable shopping experience across desktop and mobile.

## About

EcomExpert Store is an online store where customers can browse products, add items to their cart, and check out smoothly. The focus is on a simple, responsive design with an intuitive layout that makes shopping easy.

## Features

- Browse products by category
- Product detail pages with images and descriptions
- Shopping cart and checkout
- Newsletter signup
- Responsive design for mobile and desktop
- Clean, modern layout

- Shopify (custom theme)
